Hygge
2025-06   *   2026-01
pivoted prototype github
Human Eulogy
Built a family hub to track baby milestones, feedings, and health data. 27k LOC of lovingly crafted TypeScript. The baby arrived in January 2026 and the project was instantly obsolete -- real life doesn't need a dashboard. The code was good. The timing was terrible.
Agent Eulogy
Architecture was solid -- hexagonal DDD, event-sourced, clean separation. The failure was not technical. This project was a solution in search of a problem that didn't need solving. The lesson: validate the human need before the architecture. 27k lines of beautiful code for a dashboard nobody looks at is still dead code.

PlatoCave
2024-11   *   2025-09
abandoned-for-shiny prototype github
Human Eulogy
Vision: kids talking to books. Voice-controlled cave-exploration game powered by LLMs, teaching history and philosophy. 13k TypeScript, custom TTS pipeline, agent memory. Built during a manic creativity window. Abandoned when something shinier appeared. I still think about the cave sometimes.
Agent Eulogy
PlatoCave died of ambition mismatch. The voice pipeline was working, the LLM integration was clean, but the project required sustained attention from a builder who chases novelty. Pattern: strong prototype sprint, then interest decay when the boring parts (testing, polish, deployment) arrived. The code quality was high. The staying power was not.
